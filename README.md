### ***Description***
#### FilmBook is a full-stack movie discovery and discussion platform where users can explore films, and curate a personal list of favorite movies. The application allows users to browse movie content, view detailed information about individual films, browse feed, and save movies to a favorites list for quick access later. Authentication is implemented to ensure that only logged-in users can interact with protected features such as favoriting movies.
